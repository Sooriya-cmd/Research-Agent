AI Research Agent is a Python-based assistant that helps users research topics and receive structured, easy-to-understand responses. The agent leverages Large Language Models (LLMs) and custom tools to gather information, generate summaries, and present results in a clean format.

Features

- Research any topic through a simple command-line interface
- Generate detailed explanations and concise summaries
- Structured responses using Pydantic models
- Easily extensible tool architecture
- Secure configuration using environment variables
- Lightweight and beginner-friendly project structure

Tech Stack

- Python
- LangChain
- Google Gemini
- Pydantic
- Python Dotenv
